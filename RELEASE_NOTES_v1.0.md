# Release Notes — Hack-Agent v1.0

**Release Date**: 2026-06-28

## Overview

Hack-Agent v1.0 is an autonomous software engineering organization runtime. It simulates, judges, and evolves hackathon project execution across multiple dimensions: civilization, economy, judges, agents, organizations, adversarial dynamics, and deterministic reproducibility.

## New Features

### Unified Runtime OS
- Top-level orchestrator integrating all subsystems
- Phases 1–16 including Phase 13.5 (goal monitoring, convergence, resilience)
- Taste & Simplicity Governor for demo-quality enforcement
- Demo Surface Compiler for execution path collapse

### Determinism Engine
- Seeded RNG, deterministic UUIDs, deterministic timestamps
- Replay engine for exact mutation sequence reproduction
- Cross-run seed reproducibility verified (24 tests)

### Benchmark Suite
- 5 real-world winning hackathon strategy templates
- `HackathonBenchmarkRunner` with mutation testing
- Adversarial mutation mode
- Evaluation orchestrator with scoring

### CLI & Tooling
- 13 CLI commands: run, simulate, resume, status, memory, benchmark, replay, deploy, test, explain, health, chat, help
- Deterministic replay of past runs
- Organizational memory with query/stats/clear

### Civilization & Economy Systems
- Civilization evolution engine
- Resource ledger with budget management
- Economy enforcement hooks
- Market model with supply/demand pricing

### Adversarial System
- Intent engine with latent strategy vectors
- Cross-entity interference and conflict resolution
- Deception layer with false data planting
- Judge adversarial drift modeling

### Agent & Organization Evolution
- Capability evolution engine with mutation tracking
- Organization evolution with department specialization
- Swarm evolution engine
- Strategy genome database with win-rate tracking

## Changes from v0.1.0

### Major
- Complete TypeScript error resolution: 775 → 0 errors
- Test certification: all 1138 tests passing
- Static analysis: 0 lint errors (1163 warnings remain)
- Added Phase 13.5 subsystems (goal monitor, convergence, resilience, sandbox)
- Added Demo Surface Compiler for execution path collapse
- Added Taste & Simplicity Governor

### API Changes
- `computeReward()` now returns `RewardSignal` (object with `totalScore`) instead of `number`
- `StrategyPlan.techStack` is now optional
- `CivilizationDashboard` renamed to `DashboardGenerator` (class) / `DashboardData` (interface)
- Added `DepartmentType.INNOVATION` enum value
- Added `getAllAgents()`, `createAgent()`, `setEvolutionPressure()` to agent evolution engines
- Added `getAllJudges()`, `addJudge()` to `GlobalHackathonWorld`
- `InternetHackathonOrchestrator` now has `setDevpostData()` and `buildExecutionPlan()` methods

### Removed
- Duplicate re-exports from barrel files
- `require()` style imports (converted to ESM `import`)

## Known Issues

1. **OOM in CI**: 3 test files need >2GB heap. Set `NODE_OPTIONS=--max-old-space-size=4096`.
2. **1163 lint warnings**: Mostly import ordering and unused variables. Not blocking.
3. **No distributed execution tests**: Worker pool is untested in automation.
4. **No long-run verification**: 10k epoch simulations not validated in this release.
